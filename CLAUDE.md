# InScribe App Development Guide for AI SWE Agents

This is the InScribe app, a simple note-taking app made for iPad and Apple Pencil.
The features and a general overview of how to implement them can be found in the base directory in the InScribe_Spec_Sheet.md file.

When building this app, prioritize making it lightweight and minimal, while still implementing the features requested. Minimize bloat or unecessary code.

When debugging or adding new features, ALWAYS BUILD the app to make sure that there are no errors.

If you need to better understand documentation or look something up, you should do so using the Exa MCP server.
